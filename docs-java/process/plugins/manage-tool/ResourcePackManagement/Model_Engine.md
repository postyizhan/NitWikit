# Model Engine

此插件可以帮助你打包资源包生物模型

闭源付费插件，并且 v3 和 v4 分开付费，v3 用户想用 v4 得再买一次

- v3 支持 1.16 - 1.20
- v4 支持更新的 MC 版本

:::info

`插件百科` https://mineplugin.org/ModelEngine

:::
